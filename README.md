# civil-park-api

A Rust API project built with [Axum](https://github.com/tokio-rs/axum) and [Tokio](https://tokio.rs/).

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (latest stable version recommended)

## Development

To run the server locally:

```bash
cargo run
```

To build for release:

```bash
cargo build --release
```
